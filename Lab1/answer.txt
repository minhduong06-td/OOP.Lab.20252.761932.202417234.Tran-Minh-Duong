Bài 6.1 – Answers

1) Điều gì xảy ra nếu người dùng chọn “Cancel”?

Nếu người dùng chọn “Cancel”, phương thức sẽ trả về JOptionPane.CANCEL_OPTION.

Tuy nhiên, trong chương trình đã cho, đoạn mã chỉ kiểm tra giá trị trả về có phải là
JOptionPane.YES_OPTION hay không:

(option == JOptionPane.YES_OPTION ? "Yes" : "No")

Vì vậy, nếu người dùng bấm “Cancel”, chương trình vẫn sẽ hiển thị:

You've chosen: No

Nói cách khác, trong chương trình này, “Cancel” bị xử lý giống như “No”.

2) Làm thế nào để tùy chỉnh các lựa chọn cho người dùng, ví dụ chỉ còn hai lựa chọn “Yes” và “No”?

Có thể dùng phiên bản đầy đủ hơn của showConfirmDialog() và chỉ định kiểu lựa chọn là
JOptionPane.YES_NO_OPTION. Khi đó hộp thoại sẽ chỉ hiện hai nút “Yes” và “No”, không còn nút “Cancel”.

Ví dụ:

import javax.swing.JOptionPane;

public class ChoosingOption {
    public static void main(String[] args) {
        int option = JOptionPane.showConfirmDialog(
            null,
            "Do you want to change to the first class ticket?",
            "Confirmation",
            JOptionPane.YES_NO_OPTION
        );

        JOptionPane.showMessageDialog(
            null,
            "You've chosen: " + (option == JOptionPane.YES_OPTION ? "Yes" : "No")
        );

        System.exit(0);
    }
}
